# Decropper
Decrop Tool for Wav file by C++

## コマンドラインの書式
decropper {入力ファイル名} {出力ファイル名} {両側サンプル数}

## コマンドライン引数の説明
・入力/出力ファイル名：
　8bitか16bitで、モノラルかステレオのwavファイル  
・両側サンプル数：
　「割れている」(ボリューム最大値に張り付いてる)部分を補間するために使用される、
　両側それぞれにおけるサンプル数。大きいほど質が良い……はずだが体感しづらいので多くとも5ぐらいでOK。
